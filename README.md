===

A quick and dirty [Yeoman](http://yeoman.io) generator for es6 projects on NPM using babel.

## Getting Started

```bash
# if you have not already installed `yo`
npm install -g yo
```

```bash
npm install -g generator-es6-lightning
```

Finally, initiate the generator:

```bash
yo es6-lightning
```
